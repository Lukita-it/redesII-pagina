Despliegue de un Servidor Web en Fedora

Este proyecto muestra cómo instalar Apache, configurar una red en clase B y desplegar páginas web personalizadas en un entorno Linux.

Ver detalles

¿Qué incluye este proyecto?

La práctica desarrolla los pasos para configurar un servidor Fedora con Apache, verificar conectividad en red y desplegar aplicaciones web personalizadas.

Servidor Apache

Instalación y configuración de Apache en Fedora Server.

Red Clase B

Configuración de red con IP estática y pruebas de conectividad.

Sitio Web Personalizado

Diseño de página web y despliegue en servidor local y hosting externo.

Resultado esperado

Al finalizar, el servidor podrá mostrar sitios web personalizados, responder a peticiones desde la red local y contar con una versión publicada en un hosting gratuito.
